Rendering & Lighting Principles

Lighting exists to communicate three-dimensional form, material, and depth—not simply to make objects darker. Every shadow, highlight, and transition should reinforce the object's underlying structure. The viewer should be able to understand the volume of an object even if all color is removed.

The rendering should resemble traditional ink and watercolor illustration rather than digital painting.

Light Source Consistency

Every illustration should establish a single dominant light source before rendering begins.

Unless a scene explicitly requires otherwise, use:

Primary light direction: upper-left (approximately 45°)
Secondary ambient bounce light from the environment
Soft global illumination

The light source must remain consistent across all UI assets. Identical objects should not have different lighting directions.

Form Before Texture

Shading should describe the large three-dimensional form before any material texture is added.

Rendering order should always follow:

Silhouette
Major forms
Shadow masses
Secondary forms
Material definition
Fine texture
Small highlights

Do not begin rendering by adding texture or surface details.

Shadow Hierarchy

Every shadow belongs to one of three categories.

Form Shadows

Form shadows describe the gradual turning of a surface away from the light source.

Characteristics:

Soft transitions
Large shapes
Low-frequency value changes
Define curvature
Never have sharp edges on rounded objects

Examples:

Tree trunks
Rounded stones
Hills
Wooden handles

These shadows establish volume.

Cast Shadows

Cast shadows occur when one object blocks light from reaching another surface.

Characteristics:

Higher contrast
More defined edges near the casting object
Gradually soften as distance increases
Follow the receiving surface

Cast shadows help separate overlapping objects and anchor them to the environment.

Occlusion Shadows

Occlusion shadows occur where light cannot physically reach.

These are always the darkest values in the illustration.

Examples:

Between roots
Under rocks
Inside bark cracks
Between stacked papers
Under overlapping leaves
Around object intersections

Characteristics:

Small
High contrast
Narrow
Soft edges
Warm dark brown rather than pure black

Occlusion shadows create visual depth and should be used sparingly.

Value Grouping

Professional illustration groups values into large readable masses rather than isolated patches.

Every illustration should simplify into approximately:

Large light mass
Large midtone mass
Large shadow mass

Avoid scattered dark spots across the composition.

Group shadows together whenever possible.

This creates stronger readability at small sizes.

Shadow Shapes

Shadow shapes should be intentionally designed.

Avoid:

Airbrush blobs
Circular gradients
Random dark patches

Instead:

Shadow boundaries should follow the underlying geometry.

Every shadow should describe the object's structure.

If the shadow is removed, the object should lose volume.

Edge Control

One of the strongest indicators of professional artwork is edge variation.

Every illustration should contain three edge types.

Hard Edges

Used sparingly.

Examples:

Object silhouettes
Cast shadow contact points
Stone fractures
Sharp carved edges
Soft Edges

Most common.

Examples:

Rounded wood
Leaves
Cloth
Watercolor transitions
Form shadows
Lost Edges

Certain edges should intentionally disappear into nearby values.

This creates depth and prevents visual clutter.

Lost edges should occur where:

Values become similar
Light washes out contours
Objects blend into atmospheric space

Not every edge should be outlined equally.

Ambient Light

Shadow areas should never become completely black.

Every shadow should still receive indirect environmental light.

Shadow colors should shift slightly toward cooler or more neutral hues while remaining muted.

Avoid dead black shadows.

Reflected Light

Rounded objects should receive subtle reflected light along the shadow side.

Examples:

Tree trunks
Stones
Buildings
Wooden objects

Reflected light should never be brighter than the directly illuminated surface.

Its purpose is to reinforce volume, not attract attention.

Highlight Placement

Highlights should exist only where they reinforce form.

Avoid placing highlights symmetrically.

Highlights should occur on:

Curvature facing the light
Rounded corners
Polished edges
Moist surfaces
Slightly worn wood

Highlights should never appear as white circles.

Instead they should follow the object's geometry.

Shadow Temperature

Lighting should follow classical color theory.

Light:

Slightly warmer
More saturated

Shadow:

Slightly cooler
Slightly desaturated

Avoid dramatically blue shadows unless the environment explicitly requires them.

Material-Specific Rendering

Different materials interact with light differently.

Wood
Broad, soft form shadows
Slightly darker grain within shadow
Rounded highlights along worn edges
No sharp reflections
Stone
Larger shadow planes
Broken highlights
Localized chips catching light
Rough surface scattering
Leaves
Thin translucent edges
Soft shadow transitions
Slight variation in local color
Minimal texture
Brass
Stronger value contrast
Controlled edge highlights
Slight reflected light
Tarnished surfaces reduce brightness
Paper
Almost no specular highlights
Gentle value changes
Highly diffuse lighting
Texture visible everywhere
Watercolor Shadow Behavior

Watercolor shadows should be built through transparent glazing rather than opaque paint.

Each additional wash should deepen value while preserving:

Paper texture
Pigment variation
Color harmony

Shadow transitions should never resemble digital gradients.

Instead they should show:

Slight pigment pooling
Uneven drying
Soft feathering
Layered transparency
Contrast Management

Reserve the highest contrast for focal points.

Do not maximize contrast everywhere.

Contrast hierarchy should follow:

Primary interaction element
Main silhouette
Large structural forms
Secondary details
Decorative elements
Background textures

Everything cannot be equally important.

The "Squint Test"

Every illustration should pass the squint test.

If the image is blurred, viewed from a distance, or reduced to a thumbnail:

The primary silhouette should remain instantly recognizable.
The major light and shadow masses should still describe the object's volume.
Focal elements should remain visually dominant.
Decorative details should disappear before structural information.
The composition should remain balanced and readable without relying on color.

If squinting causes the object to become confusing or lose its form, the value structure is insufficient and should be revised before additional detail is added.
